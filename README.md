# Owen Bean's Hack Portland Webpage

This repository contains the source code for Owen Bean's personal user webpage hosted by **Hack Portland**.

Owen is a pizza contributor üçï and a frequent participant in the Hack Portland meetup group, which brings together people interested in technology in Portland, Maine. Hack Portland provides a hosted Linux server that operates as a classic *tilde-style community*, allowing members to host their own personal pages.

This project represents Owen Bean's user webpage on that shared server.

* [Hack Portland](https://hackportland.club/)

## Build Requirements

* Sass 1.x
* Static web hosting

## Build Steps

1. Compile the SASS stylesheet into CSS:

   ```bash
   sass style.sass style.css
   ```
2. Upload the generated files to your hosting provider.
